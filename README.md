# Sentry AI

AI-Powered Burnout-Aware Productivity Assistant

## How to Run
#### Install depends:
```bash
pip3 install -r requirements.txt 
```
#### Fill `.env`
#### Run using uvicorn
```bash
uvicorn sentry_app.main:app --host 0.0.0.0 --port 8000 --reload
```
